# 天天新鮮麵包訂購系統

## 1.會員登入註冊與訂購功能。

### 1-1:會員註冊:<br>
提供user註冊，以便使用訂購功能，根據user所提供的客戶姓名、電話、顧客編號，與帳號密碼，判斷帳號是否有跟資料庫重複，如果沒有則顯示註冊成功。<br>

### 1-2:會員登入<br>
根據資料庫的帳號與密碼，與所輸入的帳號密碼是否一致來判斷登入與否。

### 1-3:客戶訂購<br>
根據下拉式選單選擇所需要的產品與數量並按下新增，將資料先存放於右邊的表格欄位。<br>
選擇完畢後，點擊表格右下的送出，將資料送出並完成訂單入資料庫。



## 2.員工登入與產品設定功能。

### 2-1:員工登入:<br>
根據資料庫的帳號與密碼，與所輸入的帳號密碼是否一致來判斷登入與否。<br>

### 2-2:產品設定功能:
登入成功後，產品設定頁面提供4種功能。<br>
*新增產品:* 根據所輸入的產品號碼、產品名稱與產品價格來新增產品進入資料庫。<br>
*更新產品:* 根據所輸入的產品ID(private key)，輸入要修改的部分，如產品號碼，產品名稱與價格，按下送出調整資料庫功能。<br>
*刪除產品:* 根據所輸入的產品ID(private key)，送出刪除資料庫的此筆資料。<br>
